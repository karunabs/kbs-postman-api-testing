
# Postman API Testing – ReqRes

This repository contains API tests created using Postman to validate REST APIs.

## Authentication
- API key authentication using `x-api-key`
- Validated authorized and unauthorized responses

## Endpoints Tested
- GET list users
- GET single user (positive & negative)
- POST create user
- PUT update user
- DELETE user


## Test Coverage
- Status code validation
- Response time validation
- Response body structure validation
- Mandatory field validation
- Email format validation
- Negative test scenarios

## Tools Used
- Postman
- JavaScript (Postman Tests)

## How to Run
1. Import the collection JSON into Postman
2. Add your API key:
   x-api-key: <YOUR_API_KEY>
3. Send requests to execute tests
